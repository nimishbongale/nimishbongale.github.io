Accelerator-Aware Model Serving

Inference Optimization Pipeline

Built a pipeline to fine-tune and quantize LLM/SLM families and deploy them as efficient services, achieving high throughput with very large context windows for practical enterprise use cases.

Role

Senior AI Research Engineer

Models

Whisper, Phi, Llama, Mistral

Deployment

AWS SageMaker

Focus

Performance and scale

Problem

LLM workloads required improved throughput and latency without sacrificing context length, while still meeting production constraints around reliability and operational cost.

Approach

  • Fine-tuned and quantized multiple model families using ONNX, AWQ, and GPTQ methods.
  • Benchmarked model variants across throughput, memory footprint, and context-window behavior.
  • Deployed optimized serving endpoints with cloud-native orchestration and Kubernetes-based release controls.
  • Integrated observability and quality checks for safe operation under real workload patterns.

Tech Stack

ONNX Runtime AWQ GPTQ Kubernetes AWS SageMaker Python MLOps
